# Calisthenics Progression Trainer

A mobile-first React + TypeScript Vite app for adaptive upper-body calisthenics workouts. The app excludes leg training, stores settings/history/progression in `localStorage`, and uses a conservative progression engine based on post-workout feedback.

## Features

- First-launch onboarding for equipment and level.
- Adaptive workout generation for Pull, Push, Shoulders, Core, and optional Skill/Control work.
- Active workout checklist with progress indicator.
- Feedback flow: Easy, Good, Hard.
- Persistent completed workout history and progression state via `localStorage`.
- Settings screen for changing equipment/level and resetting progress.
- Tailwind CSS mobile-first gym-app styling.
